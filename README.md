# Zoorganize


"Es ist eine Anwendung zu entwickeln, die einem Zoo die Verwaltung der Tiere ermöglicht. Tiere unterschiedlicher Arten sollen angelegt werden können. Die Arten sind frei konfigurierbar. Für jedes Tier ist eine digitale Akte zu entwickeln. In dieser Akte wird auch eine Chronologie von
besonderen Ereignissen gespeichert (z. B. Datum des Zugangs, Tierarztbesuche mit Datum, Zeiträume, in denen das Tier an andere Zoos ausgeliehen war, usw...)."

"Es wird eine Planung der Räumlichkeiten / Gehege / Anlagen benötigt. Für jedes Tier sind seine Anforderungen an eine Anlage (Lebensraum) zu modellieren. Das System soll eine Zuordnung von Lebewesen zu Anlagen speichern und dabei sicherstellen, dass Lebewesen den richtigen Anlagen zugeordnet sind. "

Neben der Verwaltung von Tieren und Anlagen sind auch die Mitarbeiter*innen zu verwalten. Es ist eine Basisliste der Mitarbeiter*innen anzulegen. Für jeden Mitarbeiter ist zu speichern, welche Arten von Tieren er/sie betreuen kann. 

Bei Anlage von Arten kann der Zoo selbst bestimmen, welche Felder in der digitalen Akte eines Tiers ausgefüllt werden. Für jedes Feld kann sein Datentyp sowie die Information, ob es sich um ein Pflichtfeld handelt, gespeichert werden.
